Society Management Website Project Prompt

Create a modern, responsive, and professional Society Management Website that allows users to register as society members and enables an administrator to manage society content and publish updates. The website should have both a public-facing frontend and a secure admin dashboard.

Technology Stack
Frontend: HTML5, CSS3, JavaScript (or Angular/React)
Backend: Spring Boot (Java 21)
Database: PostgreSQL
Build Tool: Maven
Security: Spring Security with JWT Authentication
File Storage: Local storage or cloud storage for gallery images
RESTful API architecture
Core Features
Member Registration
Visitors can register as society members.
Registration form should capture:
Full Name
Email Address

Email uniqueness validation.
Secure password encryption.
Login functionality for registered members.
Member profile management.
Administrator Features
Secure admin login.
Dashboard displaying:
Total members

Recent updates
Gallery statistics
Create, edit, publish, and delete announcements.
Manage registered members.
Approve or reject membership applications.
Upload and manage gallery photos.
Manage society activities and programs.
View contact messages submitted by visitors.
